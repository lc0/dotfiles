How to use it
-------------

    cd ~
    git init
    git remote add origin git://github.com/smokey42/dotfiles.git
    git pull
    git submodule init
    git submodule update
    vim -c BundleInstall -c qa

After the last command, all referenced *vim* plugins will be installed.
